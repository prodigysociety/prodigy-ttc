About Prodigy Society

Established in 2083 B.S. at Tilottama Campus, Tilottama-2, Yogikuti, Rupandehi, Nepal.

Our Mission

Prodigy Society is dedicated to driving innovation, technical development, leadership skills, and community collaboration among students at Tilottama College.

Our Vision

Building an ecosystem that bridges academic learning with hands-on projects, preparing students to excel as technology leaders globally.
